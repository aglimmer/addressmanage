# 基于C++控制台通讯录管理项目

## 环境

Dev-C++ 5.11

## 简介

- 这是一个简易的小项目，适合入门学习C++初学者
- 其中数据的操作使用C++容器`list`,`map`来实现，数据的保存使用C++文件读写功能
- 交互界面为cmd命令行窗口，数据显示格式使用经过计算填充的格式化字符串生成的表格
- 项目使用头文件包含的的方式实现多模块组合



## 功能

基于常用联系人的基本信息进行操作，包括：联系人分组、姓名、微信、通讯地址、邮箱、手机号

适用于一个用户，有用户登录功能，能对用户密码加密

对联系人基本信息实现基本的增删改查功能，能进行格式化输出

实现简单的11位手机号码验证功能
